What is SQLMate?

SQLMate is a lightweight, native macOS application for opening, exploring, and editing SQLite database files. Whether you are a developer inspecting an app's local storage, a data analyst exploring a dataset, or just curious what's inside a .sqlite file, SQLMate gives you a fast, familiar macOS interface to get the job done.

SQLMate is built entirely with SwiftUI and AppKit — no Electron, no web views, no bloat. It feels right at home on macOS because it is made for macOS.

System Requirements

  • macOS 14 Sonoma or later
  • Apple Silicon or Intel Mac
  • SQLite 3 database files (.sqlite, .db, .sqlite3)
